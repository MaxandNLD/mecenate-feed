# Меценат Лента

## Описание

Приложение "Меценат Лента" представляет собой мобильное приложение, разработанное с использованием React Native, Expo, MobX и React Query. Оно отображает ленту постов, полученных с JSONPlaceholder API, с поддержкой пагинации, механизма "pull-to-refresh" для обновления данных, а также обработкой ошибок загрузки. При нажатии на пост пользователь переходит на экран деталей поста.

## Технологии

*   **Frontend:** React Native, Expo
*   **State Management:** MobX
*   **Data Fetching & Caching:** React Query
*   **Navigation:** React Navigation
*   **TypeScript**

## Функциональность

*   **Отображение ленты постов:** Посты загружаются с API и отображаются в виде списка.
*   **Пагинация:** Автоматическая подгрузка новых постов при прокрутке списка вниз.
*   **Pull-to-refresh:** Возможность обновить ленту, потянув экран сверху вниз.
*   **Обработка ошибок:** Информирование пользователя в случае проблем с загрузкой данных.
*   **Экран деталей поста:** Отображение полного содержимого выбранного поста.
*   **Дизайн-токены:** Единые стили для цветов, отступов и типографики.
